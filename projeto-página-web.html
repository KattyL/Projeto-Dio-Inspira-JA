Inspetor de Elementos

O Inspetor de Elementos é uma ferramenta essencial para examinar o código HTML de uma página web. Essa função pode ser acessada ao
clicar com o botão direito em uma página e selecionar "Inspecionar".

Podemos utilizar diversas funções presentes na nova aba que se abrirá, a partir dessa podemos destacar a Inspeção de Elementos,
que nos permite compreender diferentes partes do nosso código.

Outra função extremamente relevante dentro do Console nos permite verificar questões que possam estar afetando nossa página,
além disso ela permite testar comandos sem necessidade de alterar nosso código.

Estrutura HTML

A estrutura do nosso código HTML é formada por elementos e tags que irão organizar nossa página e definir uma hierarquia.

Podemos destacar os principais componentes dentro da nossa estrutura HTML:

  1. DOCTYPE: Declara o tipo de documento e a versão do HTML que está sendo usada.
    Define o padrão que o navegador deve seguir ao renderizar a página.
  2. <html>: O elemento raiz que contém todo o conteúdo da página. Geralmente inclui as tags <head> e <body>.
  3. <head>: Contém informações sobre o documento, como metadados, títulos e links para folhas de estilo externas.
  4. <title>:Define o título da página, que é exibido na aba do navegador ou na guia do navegador.
  5. <meta>:Fornece metadados sobre o documento, como codificação de caracteres, descrições e palavras-chave para mecanismos de busca.
  6. <link>:Liga o documento a folhas de estilo externas, fontes ou outros recursos.
  7. <script>:Inclui scripts JavaScript que podem adicionar interatividade e funcionalidade à página.
  8. <body>:Contém o conteúdo visível da página, como texto, imagens, links e outros elementos.
  9. <header>,<nav>,<main>,<section>,<article>,<aside>,<footer>:Tags semânticas que ajudam a estruturar o conteúdo de forma significativa, melhorando a acessibilidade e o SEO.
  10. <h1>,<h2>,...,<h6>:Títulos de diferentes níveis, usados para hierarquizar o conteúdo e indicar a importância dos elementos.
  11. <ul>,<ol>,<li>:Usados para criar listas não ordenadas (bulleted) e ordenadas (numbered).
  12. <p>:Define parágrafos de texto.
  13. <a>:Cria links para outras páginas ou recursos.
  14. <img>:Exibe imagens na página.
  15. <div>:Um elemento genérico usado para agrupar e estilizar outros elementos.
